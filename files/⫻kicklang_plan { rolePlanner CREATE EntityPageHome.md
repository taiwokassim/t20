# ⫻kicklang/plan { rolePlanner CREATE EntityPageHome CLUSTER PipelineStage SYNTHESIZE output }

KickLang executes the specified rolePlanner command to create an EntityPageHome via a clustered PipelineStage, yielding synthesized output for a home page structure.[^1][^3]

## Pipeline Execution

The rolePlanner initiates a PLAN PipelineStage with GranularityFine, creating EntityPageHome as the core node. CLUSTER aggregates related entities like RolePresets, ActionVerbs, and MetaPatterns into a navigable home cluster.[^2][^3]

## Synthesized Output

```
kicklang EntityPageHome v1.0
rolePlanner CREATE EntityPageHome CLUSTER PipelineStage
  - Nodes: RolePresets, ActionVerbCatalog, ExamplePipelines
  - Edges: references, chainsTo, synthesizes
  LINK EntityPageHome, homeFor, KickLangPlaybook
  SUMMARIZE ViewHomePage: Welcome to KickLang - Formal language for layered cognition, role-based pipelines, and knowledge graph orchestration. [file:1][file:3]
output: EntityPageHome ready - traceable, modular home for Space navigation.
```

This output serves as a reusable home page artifact, chainable into broader workflows.[^3][^4]
